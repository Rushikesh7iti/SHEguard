Smart Safety Bracelet System

AI-Powered Wearable Safety Device with Offline Emergency Alerts and Evidence Recording

The Smart Safety Bracelet System is a wearable safety solution designed to help individuals respond quickly during unsafe situations such as stalking or harassment.

The device allows users to trigger safety features using simple physical interactions on the bracelet without needing to unlock a phone.

The system also works in low or no internet environments, ensuring safety features remain functional during critical situations.

🌟 Key Features

✔ Instant emergency activation through button presses
✔ AI-based stalking detection system
✔ Automatic audio evidence recording
✔ Secure timestamp + location logging
✔ Works even without internet connectivity
✔ Bluetooth-based emergency communication

⌚ Bracelet Interaction Controls

The bracelet uses four simple interaction gestures to activate safety modules.

Interaction	Action
Single Press	System Status Check
Double Press	Emergency Alert
Long Press	AI Stalking Detection
Triple Press	Evidence Recording

This design ensures quick and intuitive operation during emergencies.

⚙️ Core System Modules

The Smart Safety Bracelet system consists of five main modules:

1️⃣ Smart Bracelet Control Module
Handles user input and activates system features.

2️⃣ AI Stalking Detection Module
Analyzes repeated proximity patterns and movement similarities to detect suspicious behavior.

3️⃣ Emergency Alert Network Module
Sends emergency alerts and location data to trusted contacts and nearby safety network users.

4️⃣ Evidence Recording Module
Automatically captures and stores critical incident evidence including:

🎙 Audio recording

📍 Location

📅 Date

🕒 Time

5️⃣ Offline Communication System
Maintains system functionality without internet using local communication methods.

🧠 AI Threat Analysis

The system evaluates behavioral patterns and generates a Threat Probability Index to estimate possible stalking risk.

Example:

Threat Probability Index: 88%
Close Range Persistence: 83%

This allows the system to identify suspicious patterns early and suggest safety actions.
